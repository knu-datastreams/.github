## 프로젝트 목표
- 데이터 처리 자동화: 다양한 출처의 금융 데이터를 자동으로 수집·정리하여 분석 효율성과 최신성 확보
- 정보 접근 격차 해소: 개인투자자도 고급 분석 자료에 접근할 수 있도록 정보 접근성 평등화
- 리포트 실시간화: 주기적 수동 작성 대신 실시간 업데이트 가능한 금융 리포트 시스템 구축
